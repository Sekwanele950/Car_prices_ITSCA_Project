# Vehicle Valuation System – Mutuka Automotive

A data-driven system that predicts pre-owned car prices based on vehicle specifications, with decision rules for when to automate pricing and when to use manual review.

---

## 📊 Key Results

| Metric | Value |
|--------|-------|
| **Best Model** | Random Forest |
| **R² Score** | 0.951 (explains 95.1% of price variation) |
| **Mean Absolute Error (MAE)** | R25,250 |
| **Auto-Pricing Coverage** | 64% of vehicles |
| **Manual Review Rate** | 36% of vehicles |

---

## 📌 About This Project

This project was completed for **Mutuka Automotive**, a pre-owned car reseller, as part of the **Scientific Computing with Python (ITSCA2-12)** module at **Eduvos** (NQF Level 6).

The system predicts car prices based on specifications (engine size, horsepower, weight, fuel type, body style, etc.). Since the dataset does not include mileage, age, or accident history, this is a **first-level valuation tool** that flags high-risk vehicles for manual review.
